# Flash Cards Quiz App

I made a clean, interactive quiz app built with HTML, CSS, and JavaScript. Made for learning, testing knowledge, and practicing smooth user flow with polished front-end design.

# Features

Multi-screen system: Start → Quiz → Result

Dynamic question loading from kb.JSON

Multiple-choice questions (A–D) with real-time scoring

Result feedback based on performance

Responsive design with hover effects and subtle animations

High performance: optimized for Lighthouse 99–100%
